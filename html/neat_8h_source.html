102 // SIGMOID FUNCTION ********************************
103 // This is a signmoidal activation function, which is an S-shaped squashing function
104 // It smoothly limits the amplitude of the output of a neuron to between 0 and 1
105 // It is a helper to the neural-activation function get_active_out
106 // It is made inline so it can execute quickly since it is at every non-sensor
107 // node in a network.
108 // NOTE: In order to make node insertion in the middle of a link possible,
109 // the signmoid can be shifted to the right and more steeply sloped:
110 // slope=4.924273
111 // constant= 2.4621365
112 // These parameters optimize mean squared error between the old output,
113 // and an output of a node inserted in the middle of a link between
114 // the old output and some other node.
115 // When not right-shifted, the steepened slope is closest to a linear
116 // ascent as possible between -0.5 and 0.5
117 extern double fsigmoid(double,double,double);
